# Shahsawar Orakzai | Portfolio

A high-performance, minimalist portfolio showcasing my expertise in **Backend Architecture, Intelligent Automation, and GenAI Systems.**

## 🚀 Live Demo
[View Live Site](https://orakzai-io-portfolio.hf.space)

## 🛠️ Tech Stack
- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Vanilla CSS (Premium Aesthetics & Responsive Design)
- **Deployment**: Docker, Nginx, Hugging Face Spaces

## ✨ Key Features
- **Modern UI**: Dark-themed, glassmorphism design with custom particle animations.
- **Optimized Performance**: Multi-stage Docker builds for rapid deployment.
- **Mobile First**: Fully responsive layout optimized for all devices.
- **SEO Ready**: Proper metadata and semantic HTML structure.

## 📦 Local Development
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 🐳 Docker Deployment
```bash
docker build -t portfolio .
docker run -p 7860:7860 portfolio
```
